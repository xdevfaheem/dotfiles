my dotfiles

## usage

### stow (create symlink)
dry run - `stow -n .`
actual stow - `stow .`

### unstow (remove symlink)
dry run - `stow -nD .`
actual stow - `stow -D .`

### restow
`stow -D .`
`stow .`

or 

`stow -R .`
